SP Styleguide

Visual design system derived from sp.nl — for building apps in the SP visual identity. Based on live CSS analysis of the production website (March 2026).

01 — Colors

The SP palette is defined by a stark contrast between SP Red (the party's primary identity color), Teal (secondary accent), and a tight greyscale. No gradients. No opacity tricks. High contrast is intentional — this is political communication.

02 — Typography

Two fonts. Headers use Helvetica Inserat (condensed, ultra-bold) set in ALL CAPS — it's the visual engine of the SP brand. Body copy uses Helvetica Neue, regular weight. Border-radius is 0px everywhere. No softness, no rounding.

11 — Design Principles
Bold & Directional

Every visual choice communicates urgency and confidence. No decorative elements — only content and colour do the work. Typography is BIG.

Zero Radius

Absolutely no border-radius on any element. Corners are sharp. This is intentional — it reflects the party's direct, uncompromising identity.

All Caps Headings

All headings and navigation are uppercase. This creates instant visual hierarchy and makes the brand feel like a political poster.

Colour as Signal

Red = action, identity, emphasis. Teal = editorial content, hero. Dark/black = high-impact CTA. White = neutral reading. Never mix signals.

Two Fonts Only

Helvetica Inserat for all display/headings, Helvetica Neue for body. No decorative fonts. No italics in UI (editorial use only).

Flat & Fast

No box-shadows on cards. No gradients. Hover transitions are 500ms — noticeable but not slow. Background colour is the primary layering tool.
